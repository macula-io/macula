K-bucket for Kademlia routing table.

Description

K-bucket for Kademlia routing table. Stores up to k nodes with LRU eviction policy.

Data Types

bucket()

bucket() = #{capacity := pos_integer(), nodes := [node_info()]}

Ordered: head = oldest, tail = most recent

node_info()

node_info() = #{node_id := binary(), address := {inet:ip_address(), inet:port_number()}, last_seen => integer()}
