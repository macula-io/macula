Local RPC procedure registration registry.

Description

Local RPC procedure registration registry. Maps URIs to handler functions. Supports multiple handlers per URI (for load balancing).

Data Types

handler_fn()

handler_fn() = fun((map()) -> {ok, term()} | {error, term()})

invocation_strategy()

invocation_strategy() = round_robin | random | local_first

registration()

registration() = #{uri := binary(), handler := handler_fn(), metadata := map()}

registry()

registry() = #{registrations := [registration()], strategy := invocation_strategy()}
