RPC call routing strategies.

Description

RPC call routing strategies. Selects which provider to use for a call (local or remote).

Data Types

provider_info()

provider_info() = macula_rpc_discovery:provider_info()

registration()

registration() = macula_rpc_registry:registration()

router_state()

router_state() = #{strategy := strategy(), round_robin_index := non_neg_integer()}

strategy()

strategy() = local_first | round_robin | random | closest
